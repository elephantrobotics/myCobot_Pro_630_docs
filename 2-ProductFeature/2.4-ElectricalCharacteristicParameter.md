## 1.1 底座接口说明
| 编号 | 接口    | 定义 | 功能         | 备注         |
|:----:|:--------------:|:---------:|:-----------------:|:----------------:|
| 1    | HDMI        | HDMI*2  |连接显示屏时使用   |       |
| 2    | Type C         | 通讯接口 | 与PC机通讯 | 开发使用 |
|   3  | RS485接口     | 485通信接口 |  支持1000000波特率，支持工业总线协议，光耦隔离  |                  |  
| 4   | 开关         | 电源开关  | 控制输入电源通断  | 带灯（通电灯亮） | 
| 5   | 电源输入接口 | DC48V输入 | DC48V 输入 |                  |
| 6    | USB2.0       | USB2.0   | 可外接设备或U盘   |                |
| 7    | USB3.0      | USB3.0*4   | 可外接设备或U盘   |                  |    
| 8    | RJ45网口        |   |规格: 千兆   |       |
| 10    | 急停接口 | STOP      | 急停回路接口 |             |   



#### 1 HDMI显示接口 ：用户可以通过连接MicroHDMI显示接口，将操作页面显示到其他设备终端

#### 2 Type C ：用于与PC端连接通讯，开发人员使用

#### 3 RS485接口：支持485通信，支持1000000波特率，支持工业总线协议，光耦隔离

#### 4 电源开关：控制总电源输入的通断，关闭时，控制器也断电

#### 5 电源输入接口：本接口与DC24V电源适配器接口连接

#### 6 USB-C接口：以串口总线标准2.0进行数据连接的接口；用户可以使用USB接口拷贝程序文件，也可以使用USB接口连接鼠标、键盘等外设

#### 7 USB3.0接口：以串口总线标准3.0进行数据连接的接口；用户可以使用USB接口拷贝程序文件，也可以使用USB接口连接鼠标、键盘等外设

#### 8 RJ45网口：千兆规格

#### 10 急停回路端子：与急停按钮盒连接，可用于控制机器人紧急停止
>  **注**: 机器人使用中必须接上急停开关，并确保急停开关回路处于连通状态。



## 1.2 底座IO接口说明
| 编号 | 接口    | 定义 | 功能         | 备注         |
|:----:|:--------------:|:---------:|:-----------------:|:----------------:|
| 9   | DC/IO接口 | 24V       | DC24V             |   DC24V输出               |
|      |                | OUT1      | 数字输出信号1~6 | 输出支持PNP和NPN模式 |
|      |                | OUT2      |                   |                  |
|      |                | OUT3      |                   |                  |
|      |                | OUT4      |                   |                  |
|      |                | OUT5      |                   |                  |
|      |                | OUT6      |                   |                  |
|      |                | GND       | GND               |                  |
| 10   | DC/IO接口 | GND       | GND               |                  |
|      |                | IN6       | 数字输入信号 1~6 |输出支持PNP和NPN模式 |
|      |                | IN5       |                   |                  |
|      |                | IN4       |                   |                  |
|      |                | IN3       |                   |                  |
|      |                | IN2       |                   |                  |
|      |                | IN1       |                   |                  |
|      |                | 24V       | DC24V             |    DC24V输入，电流通常在5-20mA之间              |


#### 9 数字输出：
规格为24V/0.2A，包括 6 个数字输出信号，可用于控制继电器，电磁阀，支持PNP和NPN两种规格，具备短路、反接、雷击、过流、过压保护。

#### 10 数字输入：
6 个数字输出信号，用于接收开关信号，如急停按钮、限位开关等，支持PNP和NPN两种规格，具备短路、反接、雷击、过流、过压保护。
